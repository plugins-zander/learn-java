# 语句

Java里的语句可分为以下六类

+ 方法调用语句：`如：System.out.println(" Hello");`
+ 表达式语句：表示式尾加上分号。`比如赋值语句：x=23;`
+ 复合语句：可以用{  }把一些语句括起来构成复合语句，  
```java
如：{   z=123+x;
           System.out.println("How are you");
        }
```
+ 空语句：一个分号也是一条语句，称做空语句。
+ 控制语句：控制语句分为条件分支语句、开关语句和循环语句。
+ package语句和 import语句：它们和类、对象有关。
