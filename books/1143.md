# 类声明

## 类声明

```java
class  类名
```

```java
//以下是两个类声明的例子。
      class People {
         …
      }
      class 植物 {
         …
      }
//如：class People”和“class 植物”称作类声明；“People”和“动物”分别是类名。
```

**class 类名** 是类的声明部分，class是关键字用来定义类，类名是java标识符

给类命名时，遵守下列编程风格(这不是语法要求,但应当遵守):
    1.如果类名使用拉丁字母，那么名字的首字母使用大写字母。
    2.类名最好容易识别、见名知意。当类名由几个“单词”复合而成时，每个单词的首字母使用大写。
